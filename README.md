# Fullstack Music Player Webapp

🎵 A fullstack web application for streaming and managing music.

## Features
- User authentication (login/register/logout)
- Frontend built with HTML, CSS, JavaScript
- Backend powered by Node.js and Express
- Database integration for users, albums, and tracks
- Responsive UI for desktop and mobile

## Installation
```bash
git clone https://github.com/AyanD-coder/Fullstack_music_player_webapp.git
cd Fullstack_music_player_webapp/app/backend
npm install
npm start
